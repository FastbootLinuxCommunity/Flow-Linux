# Flow Linux
WARNING: This is a description of version 5!
It is not available in previous versions.Flow Linux is a distribution for home use. It has its own package manager, fpkg. It may have software that you don't need, but you can remove it. It supports Windows applications, flatpak, telegram, office, and gaming software. It is very easy for beginners. It is Arch-based and has a graphical installer. In addition to Flatpak, it has the following repositories: AUR (yay), and EndeavourOS.
